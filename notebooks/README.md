# Analysis notebooks

- `01_eda.ipynb` — exploratory analysis and feature investigation.
- `02_research.ipynb` — report-style statistical analysis and conclusions.
- `archive/` — legacy experiments that are not part of the active workflow.

Run these notebooks from the `notebooks/` directory so their relative paths resolve to `../data/processed/`.

Install `requirements-dev.txt` from the repository root before running them.
